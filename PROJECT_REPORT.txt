1. PROJECT OVERVIEW
-------------------
ArbPro is an AI-powered cryptocurrency arbitrage bot that detects and exploits
price discrepancies between Binance and Bybit exchanges on the BTC/USDT pair.
The system is now fully modularized into a component-based React architecture
with a FastAPI backend utilizing real-time data oracles.

================================================================================

2. ARCHITECTURE OVERVIEW (MODERNIZED)
--------------------------------------

  [A] BACKEND (Python / FastAPI)
      - api.py              : Central API & WebSocket engine. Handles real price
                              fetching from Binance and Bybit (testnets).
      - main.py             : Application entry point.
      - predictor.py        : GRU AI Predictor (Refactored for console stability).
      - execution/trader.py : Professional execution leg with CCXT async support.

  [B] FRONTEND (Modular React / TypeScript)
      - Dashboard.tsx       : Main layout engine (New).
      - components/dashboard/
          - Header.tsx      : System status and bot controls.
          - PortfolioMetrics.tsx : Real-time balance and profit tracking.
          - OracleFeeds.tsx : Live market data and threshold slider.
          - ChartSection.tsx: Recharts-powered data visualization.
          - ExecutionLedger : Full trade history log.
          - Sidebar.tsx     : AI Logic Core (Integrated Chat).

4. KEY CHANGES IN THIS ARCHITECTURE
------------------------------------
- RECHARTS TRANSITION: Moved from custom non-functional SVG charts back to 
  robust Recharts implementation with area-spread and bar-market charts.
- COMPONENT ATOMIZATION: The monolithic Index.tsx was split into 6 
  reusable components for easier maintenance and testing.
- REAL BYBIT FETCH: api.py now attempts to fetch real Bybit prices instead
  of purely simulating them. Fallback logic remains for stability.
- GLASSMORPHISM UI: Preserved high-tech neon aesthetic across all modules.
- LLM V1.0 MIGRATION: Standardized on google-genai for cleaner, safer logic.
- ROBUST JSON PARSER: Implemented Regex-based cleaning for AI responses.

================================================================================

5. KNOWN REMAINING LIMITATIONS
-------------------------------
1. BYBIT NETWORK BLOCK: Bybit calls require a functioning SOCKS5/HTTP proxy 
   at 127.0.0.1:1080. If the proxy service is inactive, Bybit data will 
   dynamically revert to a "Simulated Feed" to prevent UI hanging.

2. TESTNET DELAYS: Latency spikes from exchange testnets (especially Binance) 
   can occasionally delay balance updates by 2-5 seconds.
