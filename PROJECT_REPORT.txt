1. PROJECT OVERVIEW
-------------------
ArbPro is a high-frequency, AI-driven cryptocurrency arbitrage platform designed
to exploit price inefficiencies between Binance and Bybit. v7.0 marks a major 
architectural leap, transitioning from a single AI predictor to a 5-Tier 
Autonomous Agent ecosystem with integrated Risk Management and Decision Auditing.

================================================================================

2. CORE V7.0 ARCHITECTURE (THE 5-TIER LLM STACK)
-----------------------------------------------

  [TIER 1] AI AGENT (OPPORTUNITY DETECTION)
           - Real-time spread analysis and trade feasibility verification.
           - Implements "Logic-First" filtering before execution.

  [TIER 2] MARKET ANALYST (INTELLIGENCE)
           - Analyzes 50-candle OHLCV data from CCXT Oracles.
           - Identifies "Market Regimes" (Trending, Ranging, Volatile).
           - Feeds sentiment and resistance data to the Chat Core.

  [TIER 3] STRATEGY ADVISOR (OPTIMIZATION)
           - Batch-processes historical trade performance from the DB.
           - Recommends dynamic spread thresholds based on win-rate.

  [TIER 4] CHAT BOT (OPERATOR INTERFACE)
           - "Senior Quantitative Analyst" persona for dashboard interaction.
           - Context-aware: Uses Tier 2 data to explain market conditions.

  [TIER 5] RISK ENGINE (CAPITAL PRESERVATION)
           - Implements Kelly Criterion for intelligent position sizing.
           - Features absolute Circuit Breakers for system safety.
